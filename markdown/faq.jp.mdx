# よくある質問 / 追加情報

*このページには、すべてのプロジェクトに適用される一般的なガイドラインです。KodamaSoundsと同じくらい厳しいわけではありませんが、サークルの仕組みを把握できるよう、お読みいただけると幸いです。*


<WrappableParagraph title="一般ルール">
以下のルールは、参加するすべての人に適用されます。
-基本的な共感力、他人を尊重する気持ちがない方は、参加しないでください。言うまでもないことですが、有害・攻撃的な態度は一切禁止です。このサークルは、包括性と多様性を重視しています。これがあなたの価値観に合わない場合は、応募をご遠慮ください。
- (必須ではありませんが、強くお勧めします） KodamaSounds / COUNTERFESTのDiscordでプロジェクトのディスカッションに参加すること。このプロジェクトはコミュニティプロジェクトですので、他のアーティストと一緒に自分のアートについて話し合うことに抵抗がないことが必要です。
- プロジェクトに関する情報を勝手に知らせないこと。
</WrappableParagraph>

<WrappableParagraph title="制作手順">
1. 作っている曲がテーマに沿っているのかを確認するために、Discordコミュニティーサーバーに参加すること、またはメール、Discord、DM（連絡先は各プロジェクトページで確認できます）で連絡をとることをお勧めします。ご遠慮なくプロジェクトチャンネルでWiPsをシェアしたり、疑問があったらお聞きください。
2. 曲が完成し次第、適切なグーグルフォームに投稿してください。クレジットのフォーマットを確認するために、監督からの連絡を受けるかもしれません。
3. マスタリングを私たちに任せたのなら、マスタリングされた楽曲にご満足いただけるかどうかを確認するために連絡させていただきます。
4. アルバムのマスタリングが完全に完了したら、リリース前後に楽曲をプロモートするための資料（アートワーク、カバーアート、ロゴ）と共に、完全にマスタリングされた曲をお送りします。
5. リリース後、利用規約に従ってください。
</WrappableParagraph>

<WrappableParagraph title="アルバム/作曲ルール">
- アルバムに収録する楽曲は、いかなるプラットフォームでも未公開であり、完全な新規/オリジナル曲である必要があります。ただし、過去に制作した楽曲を著しく異なるミックスでリメイクする場合は例外とします。
- 作品の完全なクレジットを送信してください。情報には透明性を持たせてください。正確なクレジットを作成するために、可能な限り多くの情報が必要です。詳細については、以下の適切なトピックを参照してください。
- アーティスト名は45文字以内（日本語、ローマ字両方）。
- 曲名はローマ字で60文字以内、日本語で25文字以内。
- 曲名・アーティスト名が不適切な場合、または長すぎる場合は、変更をお願いすることがあります。特殊な記号や変な文字もある程度許可します。
- 楽曲は勝手に公開しないでください。短い部分だけであれば、SNSでプレビューを投稿しても大丈夫です。適切なディスコードチャンネルでWIPを投稿してもかまいせん。アルバムに参加することを発表することはいいですが、リリースまでアルバムに関連することは一切公開しないでください。
- サンプリングは明示的でなければならない。アルバムの意図の範囲内であれば、ゲームやアニメなどからサンプリングしても大丈夫です。ほとんどの場合、商業的作品からのサンプリングは問題ないはずです。ただ、論理的で公正であるべきで、他人の作品を自分のものだと主張したり、既存の同人/インディーズアーティストからのサンプリングはしないでください。
テーマに沿って作曲してください。アルバムるのスタイルに当てはまるものであれば、何でも募集しています。もし、あなたがしてもいいことに疑問があれば、私たちにご遠慮なく聞いてください。
- プロジェクトに何曲送っていただいても大丈夫ですが、必ず厳選から応募してください。応募曲数が多ければ多いほど、クオリティに関して厳しくなります。
- 私たちは、誰でもアルバムに参加することができますが、あなたの応募をアルバムに収録するかどうかの判断に影響するかもしれない品質/選曲基準があることをご理解ください。もしご作曲を受けかねるようになったら、悪く取らないでください。ご作曲は少し改善すれば例のアルバム、あるいは別のアルバムに収録される資格があると思ったら、喜んでコミュニティとこれについて話し合って、手を貸します。
</WrappableParagraph>

<WrappableParagraph title="マスタリング">
トラックのファイナルミックスは、以下のルールに従って、その設定で書き出してください。
- WAV 44.1 kHz / 24bit。
- ヘッドルームは-6dbから-12db程度。
- Mix Busでリミッターやクリッパーを使用しないこと。

### 注意事項
- すでにプロのミキシング/マスタリングをした場合、またはご自身でプロのマスタリングをされている場合は、私たちのマスタリングエンジニアに曲を送る必要はなくなります。ただし、他の楽曲と比べて音量が高すぎたり低すぎたりする場合は、少しポストマスタリングを行わせていただきます。
- 上記のルールは、フルトラック、またはステムによるマスタリングの場合の個々のトラックにも適用されます。
- ミックスができる限り正確であることを確認してください。マスタリングのポイントは、ミキシングエラーを修正することではなく、最終的なリリースの音質を向上させることなのです。
- 最終的なミックスの音質に自信がない場合、他のアーティストやアルバムのマスタリングエンジニアのセカンドオピニオンが必要なら、私たちのディスコードコミュニティでミックスを共有することができます。
</WrappableParagraph>

<WrappableParagraph title="透明性と収益">
KodamaSoundsのアルバムと違って、COUNTERFEST RECORDSのアルバムは無料です。なぜかというと、ほとんどのアルバムはサンプリングを多用しますが、配信/流通に関してはグレーゾーンです。トラックを応募する前に、アーティストに理解していただく必要があります。できるだけ利益を生んで、アルバムのデジタル販売を行うつもりはありません。Bandcampでの販売は全て 「name your price 」に設定されており、最終的に得られる収入はボーナスとみなされ、KodamaSoundsを含むプロジェクト全体の資金として直接使用されます。ただし、CDのような物理媒体の形でリリースされるアイテムは販売するつもりです。貢献してくださったアーティストに必ずアルバム1枚を送って、販売の収入はアルバム製造の費用の補診となります。
</WrappableParagraph>
