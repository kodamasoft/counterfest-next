新しい視点への準備を。**COUNTERFEST RECORDS**は、2025年から活動する予定の[KodamaSounds](https://kodamasoft.net)のBootlegサブレーベルです。COUNTERFEST RECORDSの目的は、**無料**アルバム（コンピレーションとソロ・リリース両方）をリリースすることであり、レーベルのビジョンに沿うものであれば、自分のアルバム・プロジェクトを企画することに興味のある人なら誰でも、COUNTERFEST RECORDSの下でで無料でリリースできるようにすることです。KodamaSoundsの真面目な仕方と違って、あらゆる方法で、二次的著作物をリリースするのを恐れずに、あらゆる種類の無料アルバム・コンピレーションの制作を促すことです。また、KodamaSoundsとは異なるビジョンで、トリビュート・アルバムのためにアーティストが好きなものを作る機会をより多く与えられるように努めています。2025年以降、新しい「Tribute Album」プロジェクトはすべて、KodamaSoundsではなく、COUNTERFEST RECORDSの下で無料でリリースされます。

# COUNTERFEST RECORDSの核となるアイデアは何ですか？

- オリジナル・アルバムもトリビュート・アルバムも、利益のことを考えずに、ユニークなアルバムを作るための環境。

- ブートレッグ・リミックス文化や人気のない/実験的な音楽スタイルに由来するアルバム企画。
